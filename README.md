# GitHub Pages Site

Este site estatico foi preparado para servir como pagina publica do projeto no cadastro do TikTok for Developers.

## Arquivos

- `index.html`
- `privacy.html`
- `terms.html`
- `styles.css`

## Como publicar no GitHub Pages

1. Crie um repositorio novo no GitHub, por exemplo `autoshorts-site`.
2. Envie o conteudo desta pasta para a raiz desse repositorio.
3. No GitHub, abra:
   - `Settings`
   - `Pages`
4. Em `Build and deployment`:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` e `/ (root)`
5. Aguarde o GitHub publicar.

O link final costuma ficar assim:

```text
https://SEU-USUARIO.github.io/autoshorts-site/
```

## Links Para Usar No TikTok

Depois de publicar, use:

- `Web/Desktop URL`
  - `https://SEU-USUARIO.github.io/autoshorts-site/`
- `Privacy Policy URL`
  - `https://SEU-USUARIO.github.io/autoshorts-site/privacy.html`
- `Terms of Service URL`
  - `https://SEU-USUARIO.github.io/autoshorts-site/terms.html`

## Antes de publicar

Recomendo revisar:

- nome do produto nas paginas
- texto de contato em `privacy.html`
- texto de contato em `terms.html`

Se quiser um dominio proprio depois, basta apontar o dominio para o GitHub Pages e atualizar os links no TikTok.
